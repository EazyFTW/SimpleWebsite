# Simple Website

With this tool, you can create a simple website with java running on a port.

## Options

`port` - the port of the website.

`mappings` - a list of all mappings.



## Example Config

```{
  "port": 8888,
  "mappings": [
    {
      "method": "GET",
      "url": "/",
      "fileName": "/home.html"
    }
  ]
}
```
